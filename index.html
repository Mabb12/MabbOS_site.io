О языке программирования Mabb

Mabb - это язык программирования, созданный специально для разработки приложений в операционной системе MabbOS. Его синтаксис напоминает Basic, но с некоторыми особенностями:

  • Нет отступов и точек с запятой
  • Нет поддержки ООП (похож на C в этом плане)
  • Все команды начинаются с символа подчеркивания _
  • Команды пишутся в верхнем регистре (CAPS)
  • Минималистичный набор команд для максимальной скорости

Команды Mabb

Команда Описание Пример
_PRINT Выводит текст или значение переменной (с _V) _PRINT Hello world
_PRINT _V var
_VAR Создает переменную разных типов: _TEXT, _NUMBER, _INPUT _VAR _TEXT name Lol
_VAR _NUMBER _RANDOM lol 0 10
_VAR _INPUT answer Enter a number
_EXECUTE Начинает программу с указанной строчки _EXECUTE 1
_SECOND Ждет указанное количество секунд _SECOND 5
_IF Выполняет блок, если условие истинно. Можно указывать _INT или _STR для преобразования типов _IF lol = _INT 1
_IF name = _STR Lol
_ENDIF Заканчивает блок _IF _ENDIF
_OPERATION Выполняет арифметические операции или работает с текстом (_LEN, _COUNT, _REPLACE, _CHANGE) _OPERATION result num1 + num2
_OPERATION len str _LEN
_END Заканчивает программу _END

Полезные примеры кода

Цикл for

_VAR _NUMBER count 0
// В переменной range нужно указать, сколько раз будет выполнятся цикл for
_VAR _NUMBER range 5
// Переменная для прибавления в счетчик цифру
_VAR _NUMBER lol 1
// Сам цикл
_IF count < range
_OPERATION count count + lol
_PRINT _V count
// Тело цикла for. Сюда добавляйте или удаляйте все, что угодно кроме _EXECUTE 7
_EXECUTE 7
_ENDIF

_PRINT The end

Цикл while

// Сам цикл. Укажите сюда условие, при котором цикл перестанет выполнятся.
_IF
// Тело цикла while. Сюда добавляйте или удаляйте все, что угодно кроме _EXECUTE 7
_EXECUTE 7
_ENDIF

_PRINT The end

Функция

_VAR _NUMBER func 0
// Значение, при котором выполняется фунция
_VAR _NUMBER execution 1
// Если переменная равна 1, то функция выполняется
_IF func = execution
// Отнятие из переменной, чтобы функция не выполнялась снова.
_OPERATION func func - execution
// Тело функции. Добавьте сюда любые команды
_ENDIF

_OPERATION func func + execution
_EXECUTE 5
// Вызов функции

Задания для практики

Задача 1. Калькулятор

Создайте калькулятор, который принимает первую цифру, арифметическую операцию и вторую цифру, а после выводит ответ. Сделайте так, чтобы программа работала всегда с помощью цикла.

Задача 2. Подсчет символов

Сделайте программу, которая принимает у пользователя строку, считает сколько в ней указанных пользователем символов и выводит ответ.

Задача 3. Магический шар

Создайте магический шар, который принимает у пользователя вопрос, а после выдает случайно "yes", "no" и т.д.

Задача 4. Секундомер

Создайте секундомер, который принимает у пользователя количество секунд, и ждет это количество секунд, выводя сколько секунд прошло.

Задача 5. Игра-угадайка

Создайте игру-угадайку, где нужно угадать число от 0 до 100. Пользователь вводит число от 0 до 100 и выводится меньше число чем загаданное или же меньше. Игроку дается 10 попыток.

Задача 6. Заметки

Создайте программу для хранения заметок. Там пользователь может хранить любой текст, удалять определенную заметку или добавлять.
